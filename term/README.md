## bakera1n_loader  

### usage  
```
./bakera1n_loader --help
Usage: ./bakera1n_loader [-ahnsov] [-e <boot-args>] [-u <root_device>]
	-h, --help			        : show usage
	-a, --autoboot			    : enable bakera1n boot mode
	-e, --extra-bootargs <args>	: replace bootargs
	-u, --rootful <root_device>	: use rootful
	-s, --safemode			    : enable safe mode
	-o, --no-snapshot		    : boot rootless with non-snapshot
	-v, --verbose-boot		    : enable verbose boot
```
